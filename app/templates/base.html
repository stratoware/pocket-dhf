User Management & Version Control
User Identity

Your user identity is managed through Git configuration. The name and email shown are retrieved from your local Git settings.

Version Control

All changes to DHF data are automatically tracked and can be version controlled using Git. This ensures complete traceability of all modifications to requirements, specifications, and metadata.

Configuration

To update your user information, use Git commands:

git config --global user.name "Your Name"
git config --global user.email "your.email@domain.com"
Note: Pocket DHF integrates with your existing Git workflow for seamless collaboration and change tracking.
